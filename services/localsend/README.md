# Intro
[Localsend](https://github.com/localsend/localsend) is an open-source cross-platform alternative to AirDrop. this service consist of two sections:
- [static frontend](https://github.com/localsend/web)
- [a minimal backend](https://github.com/localsend/localsend/tree/main/server) to be used as signaling server
